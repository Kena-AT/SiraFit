Backend Architecture
1. Backend Architecture Overview
Architecture Style
SiraFit uses a modular monolith backend architecture.
Reasoning:
faster development velocity 
simpler operations 
easier debugging 
lower infrastructure complexity 
avoids premature distributed systems 
easier transactional consistency 
simpler local development 
The system is logically modular but deploys as a unified backend service initially.
The architecture separates:
API layer 
domain logic 
async workers 
AI orchestration 
scraping runtime 
sync engine 
persistence 
infrastructure concerns

25. Tradeoff Analysis

lower frontend complexity

Easy to debug.
